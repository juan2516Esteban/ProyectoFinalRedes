Una dirección IP pública es aquella que se asigna a cualquier dispositivo que se conecta directamente a Internet, como el router de casa o los servidores donde se alojan las páginas web. Esta IP es visible desde Internet y es única, lo que significa que no se repite.

Una dirección IP privada es una dirección fija que se asigna a cada dispositivo conectado a una red local o doméstica. Cada dispositivo tiene su propia dirección IP privada, pero comparten la misma IP pública del router. Las IP privadas no son accesibles desde Internet y no cambian a menos que las asignemos manualmente

Un ancho de banda, es la cantidad de datos que podemos enviar y recibir en una comunicación por unidad de tiempo. Se mide en bits por segundo (bps), kilobits por segundo (Kb/s), megabits por segundo (Mb/s) o gigabits por segundo (Gb/s). Para calcularlo, podemos usar la fórmula del ancho de banda a -3dB,Además, el Q-Factor (Factor de Calidad) también está relacionado con el ancho de banda y mide la selectividad de un filtro en torno a una frecuencia central o de resonancia
